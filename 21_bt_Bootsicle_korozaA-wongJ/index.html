Basic Bootstrap

Bootstrap is an open source toolkit that allows for easier formatting and implementation of web pages. Using bootstrap, the user can focus on the front-end rather than building the underlying structure of the page.

Containers are the most basic element in bootstrap and are required when using the Bootstrap default grid system. We see containers in action right here since there is a bit of padding from the edge of the page!

Bootstrap classifies screen sizes into extra small, small, medium, large, and extra large. Demonstrated below is bootstrap's grid functionality for displaying content. Bootstrap divides the page into 12 columns and, using col-md- the user specifies what percentage of the screen the row to take up

Nav 1

Bootstrap enables developers to create navbars without the use of unordered lists. By classifying elements as nav-items and as nav-links, they are styled and the extra markup of lists are avoided.

Nav 2

Navbars are commonly placed in non-static positions. Here at Bootsicle, we have decided to fix our navbar to the top of our page. Bootstrap allows for developers to classify elements like navbars to be fixed at positions on the page. They include "fixed-top", "fixed-bottom", and "sticky-top". As you may have noticed, navbars also support form controls and components such as search bars.

Nav 3

Navbars allow for linking to specific content on the page using id's as addresses. Much like with grids, the user can specify the navbar's behavior with various screen sizes and can using spacing and flex utility classes for controlling spacing/ alignment. In addition, there are various color schemes available to choose from.

Form Demo

Bootstrap form classes utilize the same grid system as demonstrated before. They can be organized into form-rows and form-groups. These rows and groups can be placed into the grid system using rows and columns, or they can be replaced by form-rows and regular columns.

Appropriate type attribute must be used on all inputs (eg. number for numerical information). Bootstrap's .form-control class can be used for textual form controls and includes styles for sizing and general appearance.

While all features are not demonstrated on this page, Bootstrap removes much of the headache associated with forms. Bt provides facilitates the inclusion of checkboxes, radios, help text with specific guidelines, form validation, and many more features.
