What is Foundation?

Foundation is an open-source web framework that lets developers forget about the complications of styling a website. Foundation relies on the usage of its XY Grid System to pad divs and organize the layout of the website.

Where can I get it?

You can simply link Foundation's stylesheet into your page's <head> tag. You can find it below.

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/foundation-sites@6.5.0/dist/css/foundation.min.css">

If you want complete access to all of the different settings Foundation allows you to edit, here is the download link.

XY Grid System
